## 为啥看《Clean Code》 
重新审视自己写过的一些业务代码，发现熊样儿。学习一下《代码整洁之道》。虽然启示本身不值钱，但是学习启示与案例研究中清理代码的具体决策之间的关系，对现在的我是有很大帮助的。
